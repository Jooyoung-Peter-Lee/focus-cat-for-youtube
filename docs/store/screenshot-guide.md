# Screenshot Guide

Chrome Web Store 요구사항: 1280×800 또는 640×400 PNG/JPG, 최대 5장.

---

## Shot 1 — Clean YouTube (추천 제거)

**목적:** 핵심 가치 — 산만함 제거  
**방법:**
1. youtube.com 홈 접속
2. 확장 활성화 상태에서 스크린샷
3. 사이드바, 추천 피드가 사라진 깔끔한 화면

---

## Shot 2 — Warning Banner (90% 도달)

**목적:** 부드러운 알림 UX 보여주기  
**방법:**
1. options 페이지에서 limit을 현재 시청 시간보다 약간 높게 설정 (예: 현재 9분 → limit 10분)
2. 1분 더 시청하면 배너 등장
3. 배너 + 플레이어가 함께 보이도록 스크린샷

---

## Shot 3 — Cat Overlay (Soft Mode)

**목적:** 메인 프로덕트 경험  
**방법:**
1. options에서 Soft Mode 설정
2. limit 초과 상태로 만들기 (limit을 0분으로 임시 설정)
3. 고양이 오버레이 + "See you tomorrow" 문구 스크린샷
4. 하단 dismiss 버튼 보이도록

---

## Shot 4 — Cat Overlay (Hard Mode, Pro)

**목적:** Pro 기능 하이라이트  
**방법:**
1. 라이선스 키 활성화 후 Hard Mode 선택
2. 동일하게 limit 초과 유도
3. dismiss 버튼 없는 오버레이 스크린샷
4. "Pro" 배지 또는 Hard Mode 레이블이 보이면 더 좋음

---

## Shot 5 — Options Page

**목적:** 설정의 단순함 강조  
**방법:**
1. options 페이지 전체 스크린샷
2. Daily Limit, Mode 선택, License Key 입력란이 모두 보이도록
3. 깔끔한 상태 (에러 메시지 없는 기본 상태)

---

## 촬영 팁

- 브라우저 창을 1280×800으로 설정 후 스크린샷
- Windows: Win+Shift+S 또는 스니핑 도구
- 개인정보(이메일, 채널명 등)가 화면에 보이지 않도록 주의
- YouTube 다크 모드 권장 — 고양이 오버레이가 더 잘 보임
